# 🏠 My Sweet Home - Rental Property Management Website

A modern, user-friendly web application for managing rental properties efficiently. This platform helps property managers and landlords streamline operations, track tenants, manage maintenance requests, and handle financial records.

## ✨ Features

- **Property Management**: Easily manage multiple rental properties
- **Tenant Tracking**: Keep detailed records of tenant information and lease agreements
- **Maintenance Requests**: Track and manage property maintenance tasks
- **Financial Management**: Monitor rent payments, expenses, and generate reports
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **User-Friendly Interface**: Intuitive dashboard for quick access to key information

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/seeethome321/my-sweet-home.git
   cd my-sweet-home
   ```

2. **Open the application**
   - Open `index.html` in your web browser
   - Or use a local server:
   ```bash
   python -m http.server 8000
   # Then navigate to http://localhost:8000
   ```

## 📖 Usage

### Main Dashboard
- View an overview of all your properties
- Access quick statistics on occupancy and revenue
- Monitor upcoming lease expirations

### Property Management
1. Navigate to **Properties** section
2. Click **Add New Property** to register a new rental
3. Fill in property details (address, type, number of units, etc.)
4. Save and view on the dashboard

### Tenant Management
1. Go to **Tenants** section
2. Add tenant information with lease details
3. Track payment history and important dates
4. Store contact information for quick access

### Maintenance Tracking
1. Create maintenance requests from the dashboard
2. Assign priority levels (Low, Medium, High)
3. Update status as work progresses
4. View maintenance history per property

## 💰 Financial Features

- Track monthly rent payments
- Record expenses and maintenance costs
- Generate profit/loss reports
- Export financial data for tax purposes
